## Steps to install code in chrome-extension
1. Download the code
2. Go to [chrome://extensions/](chrome://extensions/)
3. Enable the **Developer Mode**
4. Load the Code using "Load Unpacked"
5. From extension Pin the "Crypto-Calc"
